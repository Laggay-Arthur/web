Что такое CMS

Cms - "система управления контентом" (движок) - написанная PHP программистом движок для сайта, с помощью котороый вы можете управлять сайтом не зная HTML и CSS

Однако, чтобы сделать сайт с помощью CSS пондобятся услуги как программиста, так и верстальщика, и дизайнера. И капиталовлодения

Какие бывают CSS

Бывают различные системы управления контентом: для интерент-магазинов, для форумов, для блогов и т.д.

Примеры Cms

Примеры популярных Cms: Joomla,WordPress,PhpBB(для форумов)
